# Privacy Policy for Global Mouse

**Effective Date:** 2026-02-13

## 1. Introduction
Global Mouse ("we", "our", or "us") is an open-source desktop utility designed to enhance mouse scrolling functionality on Windows and macOS. We respect your privacy and are committed to protecting it.

## 2. Data Collection and Usage
**We do not collect, store, or transmit any personal data.**
* **No Personal Information:** We do not ask for or collect your name, email address, location, or any other personally identifiable information.
* **No Usage Data:** We do not track how you use the application.
* **No Analytics:** The application does not contain any third-party analytics SDKs (e.g., Google Analytics, Firebase).

## 3. Data Storage
All configuration data (such as scrolling speed, sensitivity settings, and hotkeys) is stored **locally** on your device in a JSON file (e.g., `.global_mouse_config.json`). This data never leaves your computer.

## 4. Network Access
Global Mouse does not require internet access to function. It does not connect to any remote servers, nor does it upload any data.

## 5. Contact Us
If you have any questions about this Privacy Policy, please contact us via our GitHub Issues page:
https://github.com/AouTzxc/Global-mouse/issues

# 隐私政策 (Global Mouse)

**生效日期：** 2026年2月13日

## 1. 简介
Global Mouse（以下简称“我们”）是一款开源桌面工具，旨在增强 Windows 和 macOS 上的鼠标滚动功能。我们尊重您的隐私并致力于保护它。

## 2. 数据收集与使用
**我们不收集、不存储、也不传输任何个人数据。**
* **无个人信息**：我们不会要求或收集您的姓名、电子邮件地址、位置或任何其他个人身份信息。

所有配置数据（如滚动速度、灵敏度设置和快捷键）均以 JSON 文件的形式**本地存储**在您的设备上（例如 `.global_mouse_config.json`）。这些数据永远不会离开您的计算机。

## 4. 网络访问
Global Mouse 不需要互联网连接即可运行。它不会连接到任何远程服务器，也不会上传任何数据。
